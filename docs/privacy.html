1. Verantwortlicher

Verantwortlich für diese App ist Meson Labs. Kontakt: thomashaas84@web.de

2. Zweck der App

Luna Coin ist eine private Familien-App zur Verwaltung von Aufgaben, Belohnungen, Coins, Minispielen, LunaME-Profilen und einem Hundedienstplan.

3. Verarbeitete Daten

Die App kann folgende Daten speichern: Benutzernamen innerhalb der Familie, Rollen, Coin-Stände, Aufgaben, Shop-Einträge, Kauf- und Aufgabenprotokolle, Spielstände, Highscores, LunaME-Inventar sowie Einträge im Hundedienstplan.

Es werden keine Bankdaten, Ausweisdaten oder Gesundheitsdaten verarbeitet.

4. Firebase / Firestore

Zur Synchronisierung zwischen mehreren Geräten nutzt die App Firebase Cloud Firestore von Google. Die Daten werden gespeichert, damit alle Familiengeräte denselben Stand anzeigen können.

5. Weitergabe von Daten

Die Daten werden nicht verkauft und nicht zu Werbezwecken weitergegeben. Eine Verarbeitung durch Google kann im Rahmen der Firebase-Dienste erfolgen.

6. Löschung von Daten

Daten können auf Anfrage gelöscht werden. Sende dazu eine E-Mail an thomashaas84@web.de.
