🔒 Datenschutz

Datenschutz

Luna Coin ist für Familien gedacht. Deshalb werden nur Daten verwendet, die für die Funktionen der App notwendig sind.

Welche Daten werden gespeichert?

In der App können Namen von Familienmitgliedern, Aufgaben, Coins, Belohnungen, Spielstände, Highscores und Einstellungen gespeichert werden. Je nach Nutzung können diese Daten lokal auf dem Gerät oder über Cloud Sync synchronisiert werden.

Warum werden sie gespeichert?

Die Daten werden benötigt, damit Aufgaben, Coins, Belohnungen, LunaME-Gegenstände, Minispiele und der Fortschritt innerhalb der Familie korrekt funktionieren.

Werden Daten weitergegeben?

Die Daten werden nicht verkauft und nicht für Werbung verwendet. Eine Weitergabe erfolgt nur, wenn sie technisch notwendig ist, zum Beispiel für die Synchronisierung zwischen Geräten.

Wie kann ich Daten löschen?

Daten können in der App zurückgesetzt oder gelöscht werden. Zusätzlich kann eine Löschung per E-Mail angefragt werden.
